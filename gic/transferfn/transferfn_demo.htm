Example of direct computation of the transfer function in the frequency domain using H(f) = Y(f)/U(f), where Y(f) is the DFT of y(t) and U(f) is the DFT of u(t). Neither the input or output contain noise.

The input signal u(t) of length N = 104 is filtered using a discrete approximation of an exponential low-pass filter: h(t) = (1-1/10)t-1, t = 1, ..., 100 with h(0) = 0 to create the output signal y(t).
